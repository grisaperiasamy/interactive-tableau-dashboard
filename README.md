# interactive-tableau-dashboard
Interactive Tableau dashboard analyzing business data with KPI tracking and performance insights.
# 📌 Project Overview

This project presents an interactive Tableau Dashboard designed to analyze business performance and generate actionable insights.

The dashboard visualizes key metrics, trends, and category performance to support data-driven decision-making.

This project demonstrates real-world data analysis and data visualization skills required for Data Analyst roles.

# 🎯 Business Objective

Monitor overall sales performance

Analyze regional and category performance

Track KPIs and growth trends

Identify top and low-performing segments

# 📊 Key KPIs

💰 Total Sales

📦 Total Orders

📈 Profit

📊 Profit Ratio

📅 Monthly Sales Trend
